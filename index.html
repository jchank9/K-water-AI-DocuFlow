⬇ 설치 시작 (Windows)
내려받은 KwaterDocuFlow_WebSetup.bat(약 2KB 실행기)을 더블클릭하면, 설치 중 본체(약 78MB)를 자동으로 받습니다.

설치 후 진행 순서

  1. 본체 다운로드 (자동)
  2. 압축 해제 · 설치 (자동)
  3. Claude 로그인 확인
  4. 러너 자동 실행 → 작업 화면 열림

먼저 준비할 것

  • Windows 10/11 PC
  • Claude Desktop 설치 + 로그인
  • 관리자 권한 (설치 시 1회 승인)
⚠ "Windows의 PC 보호" 경고가 뜨면
서명되지 않은 설치 파일이라 SmartScreen 경고가 나올 수 있습니다. [추가 정보] → [실행] 을 누르면 진행됩니다. (사내 배포 파일이며 안전합니다)
