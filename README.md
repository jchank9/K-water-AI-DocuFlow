# K-water AI DocuFlow — 설치 및 사용 가이드

Claude로 **PPT · 보도자료 · 보고서**를 자동 생성하는 K-water 스킬 러너입니다.
아래 순서대로 한 번만 설치하면, 이후에는 바탕화면 아이콘으로 바로 실행할 수 있습니다.

> **공유 링크**
> 👉 https://jchank9.github.io/K-water-AI-DocuFlow/

---

## 0. 먼저 준비하세요 (필수)

| 항목 | 설명 |
|------|------|
| **Windows 10/11 PC** | 일반 사무용 PC면 됩니다 |
| **Claude Desktop 설치** | [claude.ai/download](https://claude.ai/download) 에서 설치 |
| **Claude 계정** | 로그인할 Anthropic(Claude) 계정 — 설치 중 1회 로그인합니다 |
| **관리자 권한** | 설치 시 "예" 한 번 클릭 (UAC 승인) |

> Claude Desktop이 설치되어 있지 않으면 설치가 끝나도 동작하지 않습니다. **먼저 설치**해 주세요.

---

## 1. 링크 접속 → 다운로드

1. 위 **공유 링크**에 접속합니다.
2. 파란색 **[설치 시작 (Windows)]** 버튼을 클릭합니다.
3. `KwaterDocuFlow_WebSetup.bat` 파일이 다운로드됩니다.
   - 이 파일은 **약 2KB의 작은 "실행기"** 입니다. (맞습니다 — 매우 작습니다.)
   - **프로그램 본체(약 78MB)는 이 파일을 실행한 뒤(3번 단계) 자동으로 다운로드**됩니다.

---

## 2. 설치 파일 실행

1. 다운로드된 **`KwaterDocuFlow_WebSetup.bat`** 을 더블클릭합니다.
2. **"Windows의 PC 보호"** 경고가 뜨면
   → **[추가 정보]** 클릭 → **[실행]** 클릭
   *(사내 배포 파일이며 안전합니다.)*
3. **사용자 계정 컨트롤(UAC)** 창이 뜨면 **[예]** 를 클릭합니다.

> 💡 검은 창이 잠깐 떴다 사라지는 것은 정상입니다 (관리자 권한으로 다시 실행됨).
> [예]를 누르면 새 검은 창이 열리고, **설치가 끝날 때까지 그 창은 닫지 마세요.**

---

## 3. 자동 설치 (진행 화면)

- 잠시 후 **브라우저에 설치 진행 화면**이 열립니다.
- 다음 단계가 자동으로 진행됩니다:
  1. 본체 다운로드
  2. 압축 해제
  3. 앱·스킬 설치
  4. Claude 확인 · 로그인
  5. 러너 실행
- 진행바가 **100%** 가 되면 설치 완료이며, 작업 화면이 자동으로 열립니다.

---

## 4. Claude 로그인 (그 PC에서 최초 1회만)

설치 직후, 그 PC에서 Claude에 **처음 로그인**해야 합니다.

- **별도의 검은 창**(제목에 *"Claude Login"*)이 자동으로 열립니다.
- 그 창의 안내에 따라 진행하세요:
  1. 로그인 방식을 묻는 경우 **Claude 계정**을 선택
  2. **브라우저가 열리면 Claude(Anthropic) 계정으로 로그인 → 권한 승인**
  3. *"Login done"* 같은 메시지가 보이면 로그인 완료
- 로그인 후 **작업 화면(브라우저 탭)으로 돌아가 `F5`(새로고침)** 를 누르세요.

> 이미 Claude에 로그인된 PC라면 이 단계는 자동으로 넘어갑니다.

### 🔁 로그인이 안 되거나 창을 놓쳤을 때 → `Claude_로그인.bat`
- 설치가 끝나면 **바탕화면에 `Claude_로그인.bat` 파일이 자동으로 만들어집니다.** (따로 설치할 필요 없습니다.)
- 로그인 창을 못 봤거나, 작업 화면에 *"Claude 미로그인"* 오류가 나면 → **바탕화면의 `Claude_로그인.bat` 을 더블클릭**하세요.
- 열린 창의 안내대로 로그인하면 됩니다. 완료 후 작업 화면에서 **`F5`(새로고침)**.

---

## 5. 사용하기

작업 화면(`http://localhost:5000`)에서 원하는 메뉴를 선택해 생성합니다.

| 메뉴 | 결과물 |
|------|--------|
| **PPT** | 발표자료 PPTX |
| **보도자료** | HWPX 보도자료 |
| **보고서** | HWPX 실무 보고서 |

제목·부서·내용(또는 PDF 첨부)을 입력하고 실행하면, 잠시 후 결과 파일이 생성되고 폴더 열기 링크가 표시됩니다.

---

## 6. 다음 실행부터

- **검은 콘솔 창을 닫으면** 러너가 종료됩니다.
- 다시 사용하려면 **바탕화면의 `KwaterSkillRunner.exe`** 를 더블클릭하세요.
  (설치는 다시 할 필요 없습니다. 브라우저 작업 화면이 자동으로 열립니다.)

## ❓ 자주 묻는 문제 (FAQ)

**Q. 검은 창이 깜빡이고 바로 사라져요.**
A. 관리자 권한 요청(UAC)에서 **[예]** 를 누르지 않았을 수 있습니다. 파일을 **마우스 오른쪽 클릭 → [관리자 권한으로 실행]** 해 보세요.

**Q. 작업 화면에서 "Claude 미로그인" 오류가 나요.**
A. 4번 로그인을 완료하지 않은 상태입니다. 바탕화면 **`Claude_로그인.bat`** 으로 로그인 후, 작업 화면에서 **F5** 를 누르세요.

**Q. 로그인 창에서 어떤 계정으로 로그인하나요?**
A. 본인의 Claude(Anthropic) 계정으로 로그인하면 됩니다. (계정 정책은 부서 안내를 따르세요.)

**Q. "claude.exe 를 찾을 수 없습니다" 라고 나와요.**
A. **Claude Desktop이 설치되어 있지 않거나 한 번도 실행한 적이 없는** 경우입니다. Claude Desktop을 설치·실행한 뒤 다시 시도하세요.

**Q. 다운로드/설치가 중간에 멈춰요.**
A. 사내망 상태에 따라 다운로드(78MB)가 느릴 수 있습니다. 잠시 기다리거나, 설치 파일을 다시 실행해 주세요.

**Q. 설치 로그는 어디 있나요?**
A. 문제 발생 시 `%TEMP%\KwaterDocuFlow_setup.log` 파일을 담당자에게 전달하면 원인 파악에 도움이 됩니다.
   (탐색기 주소창에 `%TEMP%` 입력 → 해당 파일)
